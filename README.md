# react-tutorials-animals
A small tutorial to learn and practice react development. 

Functionality:
- This app adds a random animal's picture to the view everytime "Add Animal" button is clicked. 
- Each animal-image has an 'heart' image inside it, which grows everytime the animal is clicked. 

Learnings: 
- Using event handlers
- Using state
- Importing static images 
- Adding inline and custom CSS to react components 

Start the app using `npm run dev`
